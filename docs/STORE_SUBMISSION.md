# WinClean - Microsoft Store Submission Guide

## Prerequisites

1. **Microsoft Partner Center Account**
   - Register at https://partner.microsoft.com/
   - Individual: ~$19 USD (one-time)
   - Company: ~$99 USD (one-time)

2. **Visual Studio 2022**
   - Workload: ".NET desktop development"
   - Component: "Windows 10 SDK (10.0.22621.0)"
   - Component: "MSIX Packaging Tools"

3. **Privacy Policy URL**
   - Create a simple privacy policy page (e.g., GitHub Pages)
   - Example content: "WinClean does not collect, store, or transmit any personal data."

---

## Build MSIX Package

### Option A: Using build script
```
build-msix.bat
```
Run from "Developer Command Prompt for VS 2022"

### Option B: Using Visual Studio
1. Open `WinClean.sln` in Visual Studio
2. Right-click `WinClean.Package` project → "Publish" → "Create App Packages"
3. Select "Microsoft Store (new app name)" or "Sideloading"
4. Follow the wizard

---

## Store Submission Steps

1. **Reserve App Name**
   - Partner Center → Apps and Games → New product → App
   - Reserve "WinClean"

2. **Update Package Identity**
   - After reserving, Partner Center provides:
     - Package/Identity/Name
     - Package/Identity/Publisher
   - Update these in `src/WinClean.Package/Package.appxmanifest`:
     ```xml
     <Identity Name="YOUR_RESERVED_NAME"
               Publisher="CN=YOUR_PUBLISHER_ID" ... />
     ```

3. **Create Submission**
   - Upload `.msixupload` file from `src/WinClean.Package/AppPackages/`
   - Fill in:
     - **Description** (per language)
     - **Screenshots** (at least 1, recommended 4-5, size 1920x1080)
     - **Privacy policy URL**
     - **Age rating** (complete IARC questionnaire)
     - **Category**: Utilities & Tools

4. **Certification Notes**
   - Explain `broadFileSystemAccess` usage:
     "WinClean scans user-selected drives to find large files, duplicate files, and analyze disk usage. File system access is required for the core functionality of the disk cleaning tool."
   - Explain `allowElevation` usage:
     "Administrator privileges are optional and only requested when the user explicitly clicks 'Restart as Admin' to clean protected system files."
